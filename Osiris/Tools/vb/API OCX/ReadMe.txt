ActiveX Controls

Ensure that you have the VB5 (SP3) run-time files installed!

To register the controls type:

REGSVR32 <name of OCX file>

to unregister the controls type:

REGSVR32 -u <name of OCX file>

Then use the file in VB4-32 under Add-ins or VB5 under the same name...
